Attributes

Common attributes.

Description

The info element contains meta-information about the element that contains it.

Processing expectations

Suppressed. Many of the elements in this wrapper may be used in presentation, but they are not generally printed as part of the formatting of the wrapper. The wrapper merely serves to identify where they occur.

Parents

These elements contain info: abstract, annotation, bibliolist, blockquote, calloutlist, caution, constraintdef, equation, glosslist, important, itemizedlist, legalnotice, msg, msgexplan, msgmain, msgrel, msgset, msgsub, note, orderedlist, personblurb, procedure, productionset, qandadiv, qandaentry, qandaset, revhistory, segmentedlist, sidebar, step, taskprerequisites, taskrelated, tasksummary, tip, toc, variablelist, warning.
